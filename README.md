# CSU-Library-Book

中南大学图书馆座位预约

`seats.csv` 中的数据是 2026-04-24 抓取的，更新数据可以在 `main.py` 中取消最后的注释。因为懒得写 `config.ini` 文件，所以是在 `config.py` 中修改**学号**、**密码**、**想要预约的座位**。

```
USERNAME = '学号'
PASSWORD = '密码'
SEATS = ["XF2A001","XF2A002","XF2A003",]
```
